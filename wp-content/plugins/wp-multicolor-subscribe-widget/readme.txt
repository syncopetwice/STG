== Description ==

WP Multicolor Subscribe Widget : wordpress Multicolor Subscribe Widget Used to Generate Subscritption box in sidebar ,having option to chnage it color with interative color picker.
So that you can easily change subscription box color scheme according to your Theme colors.

Installation Guide:
 
1.Upload the  plugin to your blog and just activate it.

2.Goto your Appearance>Widgets and then Edit its data as per requirement.
